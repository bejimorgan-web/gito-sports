# UI/UX Guidelines

GiTO Live Sports should look and behave like a professional sports operations platform.

## Desktop Experience

- Prioritize tables, timelines, filters, and clear state labels.
- Use restrained color for status and urgency.
- Keep primary actions visible near the operator context.
- Avoid marketing-style hero layouts inside the administration app.
- Prefer compact, repeatable layouts over decorative cards.

## Status Language

- Draft: not ready for review.
- Pending Review: waiting for operator decision.
- Approved: visible to delivery clients.
- Rejected: blocked with reason.
- Suspended: temporarily unavailable.

## Navigation

Primary desktop sections:

- Dashboard
- Schedule
- Sports
- Competitions
- Teams
- IPTV Providers
- Stream Sources
- Approvals
- Settings
