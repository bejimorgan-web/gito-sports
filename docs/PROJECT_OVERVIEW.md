# Project Overview

GiTO Live Sports is a professional live sports streaming and sports management platform. The MVP connects IPTV providers, operator stream review, sports competition organization, match scheduling, and mobile stream delivery.

## Product Areas

- Desktop administration app: the primary control center for operators.
- Mobile streaming app: the viewer-facing live stream experience.
- Backend API: lightweight service for data access, scheduling, approvals, and stream metadata.
- SQLite database: local operational persistence for the MVP.

## MVP Outcomes

- Operators can organize sports, competitions, teams, and matches.
- Operators can configure IPTV providers and stream sources.
- Operators can preview streams before publication.
- Operators can approve or reject streams for a scheduled match.
- Mobile clients can access approved live stream metadata.

## Non-Goals

- Large distributed infrastructure.
- Complex event sourcing.
- Multi-region deployment.
- Payment, subscription, and advertising systems.
- Full automated sports data ingestion.
