# Roadmap

## Phase 0: Foundation

- Create repository structure.
- Document product, architecture, and data model.
- Establish TypeScript workspaces.
- Prepare desktop, backend, mobile, and shared package boundaries.

## Phase 1: Operational MVP

- Implement SQLite schema and migrations.
- Implement sports, competitions, teams, and matches.
- Implement IPTV provider and stream source management.
- Implement stream assignment and approval workflow.
- Implement desktop preview UI.

## Phase 2: Mobile Delivery

- Scaffold Flutter app.
- Implement approved match listing.
- Implement stream playback.
- Add mobile-safe API endpoints.
- Add basic authentication flow.

## Phase 3: Reliability

- Add audit logging for approval actions.
- Add stream health checks.
- Add backup and restore workflow for SQLite.
- Improve error handling and operator feedback.
